There are several steps involved in creating a new project here is the list

First off;
Create a new folder under /application, name it the genre of game for example "mafia"
Create a new modules folder, this will contain all your modules you build specifically for this game
Create a public folder, this will hold the views specifically for this game
Create the additional folders /public/templates, /public/css, /public/js

Sample project using a "mafia" theme
Create folder /application/mafia
Create folder /application/mafia/public
Create folder /application/mafia/public/templates
Create folder /application/mafia/public/css
Create folder /application/mafia/public/js
